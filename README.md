Simple-Archiver
===================

## Task

We had to program a simple zip like archiver in our second semester operating systems course. 

## Installation

>make all

>make clean

>make test

## Commands

**New archive**

>-n ArchiveName

**Add file to archive**

>-a ArchiveName FileName

**List content of archive**

>-d ArchiveName
